# Breathing App

A simple Flutter app for guided breathing exercises using BLoC state management.

### Features

- Guided breathing with phases: Get Ready → Breathe In → Hold → Breathe Out → Hold
- Adjustable rounds and durations
- Light and dark mode support
- Responsive layout for web and mobile
- Pause/Resume functionality
